Булевы значения в Python это две константы True и False.

В Python истинными и ложными значениями считаются не только True и False.

    истинное значение:
        любое ненулевое число
        любая непустая строка
        любой непустой объект
    ложное значение:
        0
        None
        пустая строка
        пустой объект

>>> w = 'abc' - задаем строковую переменную

2. Разобраться с различиями между input() и raw_input(). Также в контексте разных версий python: 2 и 3.
В python2:
    функция input() - принимает то, что ввел пользователь и пробует запустить это как код.
    функция raw_input() - принимает то, что ввел пользователь в виде строки.
В python3: 
    функция raw_input() была переименована в input(), теперь чтобы получить от input() тот-же результат что был в Python2 необходимо ее вложить в функцию eval()
>>> n = 123
>>> n
123
>>> eval(input())
n
123
>>>
